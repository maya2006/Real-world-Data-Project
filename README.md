# 🩺 Smart Diabetes Risk Prediction System

A final-year Data Science and Machine Learning project for healthcare analytics. This system predicts diabetes risk using the **Pima Indians Diabetes Dataset** and provides personalized lifestyle recommendations via an interactive, dark-themed **Streamlit** dashboard.

---

## 🎯 Key Features

* **Advanced Preprocessing:** Handles zero-value data anomalies using strategic median imputation.
* **Exploratory Data Analysis (EDA):** Generates 6+ interactive statistical charts for data distribution.
* **Ensemble Modeling:** Trains, evaluates, and compares three separate ML models.
* **Smart Risk Classification:** Categorizes patient risk into dynamic bands (Low, Medium, High).
* **Actionable Recommendations:** Delivers automated, data-driven lifestyle advice based on metrics.
* **Production-Ready UI:** Feature-rich dashboard equipped with a dynamic risk gauge and dataset explorer.

---

## 📦 Tech Stack

* **Language:** Python 3.9+
* **IDE:** Visual Studio Code
* **Machine Learning:** Scikit-learn
* **Data Processing:** Pandas, NumPy
* **Data Visualization:** Matplotlib, Seaborn
* **Web Dashboard:** Streamlit
* **Model Serialization:** Pickle (`.pkl`)
